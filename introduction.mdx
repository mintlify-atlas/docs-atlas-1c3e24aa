## What is GestorPro?

GestorPro is a modern business management platform designed for managing projects, clients, and AI-powered chatbots from a single dashboard. Built as a learning project with Gemini AI, it demonstrates enterprise-grade patterns including multi-tenant architecture, token-based authentication, and seamless Google Sheets integration.

## Architecture Overview

GestorPro uses a serverless architecture powered by Google Apps Script as the backend, with React and TypeScript on the frontend.

<Note>
  GestorPro was developed as a study project to explore AI integration patterns with Gemini AI and demonstrate modern SaaS architecture principles.
</Note>

### Technology Stack

- **Frontend**: React 19, TypeScript, Vite, Recharts
- **Backend**: Google Apps Script (JavaScript)
- **Database**: Google Sheets (multi-tenant)
- **Authentication**: Custom token-based system with WhatsApp/Email verification
- **Bot Integration**: BuilderBot API for WhatsApp chatbots
- **Styling**: Utility-first CSS with responsive design
