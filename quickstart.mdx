# Quickstart Guide

This guide will help you get started with GestorPro, from creating your account to managing your first project.

## What is GestorPro?

GestorPro is a multi-user management panel for organizing clients, projects, and AI-powered bots. Built with React and powered by Google Apps Script as a backend, it provides a seamless way to manage your business operations.

<Note>
  GestorPro uses Google Sheets as a database and Google Apps Script for authentication and data management.
</Note>

## Getting Started

<Steps>
  <Step title="Access GestorPro">
    Navigate to your GestorPro application URL. You'll be greeted with the login screen.
    
    The authentication system supports both WhatsApp numbers and email addresses for login.
  </Step>

  <Step title="Create Your Account">
    If you're a new user, enter your WhatsApp number or email address and click **Continuar**.
    
    <CodeGroup>
    ```tsx Auth Flow
    // GestorPro checks if you're an existing user
    const handleLoginSubmit = async (e: React.FormEvent) => {
      e.preventDefault();
      const data = await callAuthScript('findUser', identifier);
      if (data.userExists) {
        setStage('verify');
      } else {
        setStage('register'); // New user registration
      }
    };
    ```
    </CodeGroup>
    
    Complete the registration form with your information:
    - **Nombre** (First Name)
    - **Apellido** (Last Name)
    - **WhatsApp** (Phone Number)
    - **Email**
    - **FechaNacimiento** (Date of Birth)
    
    <Warning>
      Make sure to use a valid WhatsApp number or email address - you'll receive a verification code there.
    </Warning>
  </Step>

  <Step title="Verify Your Identity">
    After registration, you'll receive a 6-digit verification code via your chosen contact method.
    
    Enter the code in the verification screen. The code expires in 10 minutes.
    
    ```tsx
    // Verification handler
    const handleVerifySubmit = async (e: React.FormEvent) => {
      e.preventDefault();
      const data = await callAuthScript('verifyToken', { 
        identifier, 
        token 
      });
      if (data.authenticated && data.id_sheet) {
        onLoginSuccess({ id_sheet: data.id_sheet });
      }
    };
    ```
  </Step>

  <Step title="Create Your First Client">
    Once logged in, navigate to the **Clientes** section from the sidebar.
    
    Click **Nuevo Cliente** and fill in the client information:
    ```tsx
    interface Client {
      id: string;
      name: string;           // Company or client name
      contactPerson: string;  // Contact person's name
      email: string;          // Email address
      phone: string;          // Phone number
    }
    ```
    
    <Card title="Pro Tip" icon="lightbulb">
      Use the **Client Search** feature to quickly find existing clients by phone number before creating duplicates.
    </Card>
  </Step>

  <Step title="Create Your First Project">
    Navigate to **Proyectos** and click **Nuevo Proyecto**.
    
    Fill in the project details:
    
    <CodeGroup>
    ```tsx Project Structure
    interface Project {
      name: string;        // Project name
      clientId: string;    // Select from your clients
      startDate: string;   // Project start date
      endDate: string;     // Project end date
      budget: number;      // Budget in dollars
      status: ProjectStatus; // Project status
    }
    ```
    
    ```tsx Project Status Options
    enum ProjectStatus {
      NotStarted = 'No Iniciado',
      InProgress = 'En Progreso',
      Completed = 'Completado',
      OnHold = 'En Pausa',
    }
    ```
    </CodeGroup>
    
    After creating the project, click on it to access the detail view where you can:
    - Add notes
    - Save important links
    - Store AI prompts and responses
  </Step>

  <Step title="Explore the Dashboard">
    Return to the **Dashboard** to see an overview of:
    - Total number of projects, clients, and bots
    - Project status distribution
    - Recent activity
    
    The dashboard automatically updates as you add more data to your system.
  </Step>
</Steps>

## Common Operations

### Searching for Clients

Use the client search feature to quickly find existing clients:

```typescript
// Search by phone number
import { getClientByPhone } from './services/api';

const client = await getClientByPhone('5493624...');
if (client) {
  console.log('Client found:', client.name);
}
```

### Managing Project Data

All project data is automatically synced with Google Sheets:

```typescript
import { getProjects, addProject, updateProject } from './services/api';

// Fetch all projects
const projects = await getProjects();

// Create a new project
const newProject = await addProject({
  name: 'Website Redesign',
  clientId: 'client-123',
  startDate: '2024-01-01',
  endDate: '2024-03-31',
  budget: 5000,
  status: ProjectStatus.InProgress
});
```

<Note>
  All API calls automatically include your session ID for authentication. The session is stored in localStorage and persists across page reloads.
</Note>

## Troubleshooting

### Cannot Connect to Backend

If you see a "Failed to fetch" error:

1. Verify your Google Apps Script URLs are correctly configured
2. Check that the deployment has "Anyone" access
3. Test the script URL directly in your browser

<Warning>
  Every time you update your Google Apps Script code, you must create a new deployment and update your environment variables.
</Warning>

### Session Expired

If your session expires, you'll be automatically redirected to the login screen. Simply log in again with your credentials.
