Projects are the central hub of GestorPro, allowing you to organize your work with clients and keep all related information in one place.

## Creating a New Project

<Steps>
  <Step title="Navigate to Projects">
    Click on the "Proyectos" section in the main navigation to view all your projects.
  </Step>
  
  <Step title="Click New Project">
    Click the "Nuevo Proyecto" button in the top right corner to open the project creation form.
  </Step>
  
  <Step title="Fill in Project Details">
    Enter the following information:
    - **Project Name**: A descriptive name for your project
    - **Client**: Select a client from the dropdown (clients must be added first)
    - **Start Date**: The project's start date
    - **Description**: Optional details about the project
  </Step>
  
  <Step title="Save the Project">
    Click "Guardar" to create the project. The new project will appear in your projects grid.
  </Step>
</Steps>

<Note>
Projects are automatically sorted by start date, with the most recent projects appearing first.
</Note>

## Viewing Project Details

To view detailed information about a project:

1. Click on any project card in the grid view
2. A side panel will slide in from the right showing the project details
3. The detail view displays three tabs: **Notes**, **Links**, and **Prompts**

<Tip>
On mobile devices, the detail view will take up the full screen. On desktop, it appears as a sidebar allowing you to see both the project list and details simultaneously.
</Tip>

## Editing a Project

<Steps>
  <Step title="Open the Edit Dialog">
    Click the "EDITAR" button on the project card you want to modify.
  </Step>
  
  <Step title="Update Project Information">
    Modify any of the project fields:
    - Project name
    - Associated client
    - Start date
    - Description
  </Step>
  
  <Step title="Save Changes">
    Click "Guardar" to update the project with your changes.
  </Step>
</Steps>

## Managing Project Notes

Notes help you keep track of important information, decisions, and updates for each project.

### Adding a Note

<Steps>
  <Step title="Select a Project">
    Click on a project card to open the detail view.
  </Step>
  
  <Step title="Navigate to Notes Tab">
    Ensure the "Notas" tab is selected (it's the default view).
  </Step>
  
  <Step title="Fill in Note Information">
    In the "Añadir Nueva Nota" form:
    - Enter a title for the note
    - Add the note content in the text area
  </Step>
  
  <Step title="Save the Note">
    Click "Guardar Nota" to add the note to the project.
  </Step>
</Steps>

### Pinning Important Notes

You can pin notes to keep them at the top of your list:

- Click the **thumbtack icon** on any note to pin or unpin it
- Pinned notes appear at the top with an indigo border
- Notes are sorted by pin status first, then by creation date

<Tip>
Use pinned notes for critical information that you need to reference frequently, like client requirements or important deadlines.
</Tip>

### Editing and Deleting Notes

- **Edit**: Click the **pencil icon** to load the note into the edit form
- **Delete**: Click the **trash icon** and confirm the deletion

<Warning>
Deleting a note is permanent and cannot be undone. Always double-check before confirming deletion.
</Warning>

## Managing Project Links

Store and organize important URLs related to your project, such as design files, documentation, or client resources.

### Adding a Link

<Steps>
  <Step title="Open the Links Tab">
    Click on a project to open the detail view, then select the "Links" tab.
  </Step>
  
  <Step title="Enter Link Information">
    In the "Añadir Nuevo Link" form:
    - **URL**: Enter the full URL (must include https:// or http://)
    - **Description**: Optional description to explain what the link is for
  </Step>
  
  <Step title="Save the Link">
    Click "Guardar Link" to add the link to the project.
  </Step>
</Steps>

### Using Links

- Click on any link URL to open it in a new browser tab
- Pin important links using the **thumbtack icon** to keep them at the top
- Edit or delete links using the **pencil** and **trash** icons

<Note>
Links with a description will display it below the URL for easy reference.
</Note>

## Managing Project Prompts

Prompts allow you to store AI prompts and their responses, helping you maintain a knowledge base for your project.

### Adding a Prompt

<Steps>
  <Step title="Open the Prompts Tab">
    Click on a project to open the detail view, then select the "Prompts" tab.
  </Step>
  
  <Step title="Enter Prompt Information">
    In the "Añadir Nuevo Prompt" form:
    - **Prompt**: Enter the AI prompt or question
    - **Response**: Paste the AI's response (optional but recommended)
  </Step>
  
  <Step title="Save the Prompt">
    Click "Guardar Prompt" to store the prompt and response.
  </Step>
</Steps>

### Viewing Prompt Details

- Prompts are displayed in a collapsible format
- Click on any prompt to expand it and view the full response
- The prompt text is shown as a summary, with the response hidden until expanded

<Tip>
Use prompts to document your conversations with AI tools, saving useful responses for future reference or to share with your team.
</Tip>

## Deleting a Project

<Warning>
Deleting a project will permanently remove the project and ALL associated notes, links, and prompts. This action cannot be undone.
</Warning>

<Steps>
  <Step title="Click Delete">
    Click the "ELIMINAR" button on the project card you want to remove.
  </Step>
  
  <Step title="Confirm Deletion">
    A confirmation dialog will appear. Read the warning message carefully.
  </Step>
  
  <Step title="Confirm">
    Click "Sí, Eliminar" to permanently delete the project and all its data.
  </Step>
</Steps>

## Frequently Asked Questions

<Accordion title="Can I change the client associated with a project?">
Yes, you can edit any project and change the associated client by selecting a different client from the dropdown menu.
</Accordion>

<Accordion title="How many notes, links, or prompts can I add to a project?">
There's no hard limit on the number of notes, links, or prompts you can add. However, for optimal performance, consider archiving older projects when they're no longer active.
</Accordion>

<Accordion title="Can I export my project data?">
Currently, project data is stored in your GestorPro account. Export functionality may be available in future updates.
</Accordion>

<Accordion title="What happens to projects when I delete a client?">
When you delete a client, all projects associated with that client are also deleted. Make sure to back up any important information before deleting a client.
</Accordion>

<Accordion title="Can I reorder my notes, links, or prompts manually?">
Items are automatically sorted by pin status and creation date. Use the pin feature to keep important items at the top of the list.
</Accordion>
