## Bot Data Structure

Each bot contains the following information:

```typescript
interface Bot {
  id: string;                    // Unique identifier
  name: string;                  // Bot display name
  description: string;           // Bot purpose/description
  systemInstruction: string;     // AI system prompt
  createdAt: string;             // Creation timestamp
  id_builderBot: string;         // BuilderBot platform ID
  status: string;                // Current connection status
  updatedAt: string;             // Last sync timestamp
  onlineSince?: string;          // When bot went online
  lastOnlineDuration?: string;   // Last active duration
  builderBotApiKey?: string;     // API authentication key
}
```

<Info>
  The `systemInstruction` field defines the bot's personality and behavior using AI prompts.
</Info>

## Bot Status

Bots can be in various states:

<Tabs>
  <Tab title="Active States">
    ### Online and Connected
    
    - **ONLINE / ACTIVE**: Bot is connected and responding to messages
    - **CONNECTING**: Bot is in the process of connecting to WhatsApp
    - **READY_TO_SCAN**: QR code is ready for scanning
    
    <Note>
      When online, bots display a real-time uptime timer showing how long they've been active.
    </Note>
  </Tab>
  
  <Tab title="Inactive States">
    ### Offline and Error States
    
    - **NOT_FOUND**: Bot has no active connection
    - **FAILED**: Connection attempt failed
    - **ERROR**: Bot encountered an error
    - **DISCONNECTING**: Bot is shutting down
    - **PENDING SYNC**: Bot is synchronizing with the platform
    
    <Warning>
      Bots in error states should be reconnected or reconfigured.
    </Warning>
  </Tab>
</Tabs>

## Managing Bots

<Steps>
  <Step title="Create Bot">
    Click **"Nuevo Bot"** and fill in:
    - Bot name
    - Description
    - System instruction (AI prompt)
    - BuilderBot ID
    - API key (optional)
  </Step>
  
  <Step title="Connect to WhatsApp">
    After creation, click **"Conectar"** to start the connection process.
  </Step>
  
  <Step title="Scan QR Code">
    When status becomes "READY_TO_SCAN", scan the displayed QR code with WhatsApp.
  </Step>
  
  <Step title="Bot Goes Online">
    After successful scan, the bot status changes to "ONLINE" and starts responding to messages.
  </Step>
</Steps>

```typescript
useMemo(() => {
  const intervalId = setInterval(() => {
    fetchAndProcessBots(false); // Background polling
  }, 5000); // Every 5 seconds
  
  return () => clearInterval(intervalId);

```

<Info>

## Online Timer

When bots are active, a live timer shows uptime:

```typescript
const OnlineTimer: React.FC<{ onlineSince: string }> = ({ onlineSince }) => {
  const [duration, setDuration] = useState(
    Date.now() - new Date(onlineSince).getTime()
  );
  
  // Updates every second
  useEffect(() => {
    const timer = setInterval(() => {
      setDuration(Date.now() - new Date(onlineSince).getTime());
    }, 1000);
    return () => clearInterval(timer);
  }, [onlineSince]);
  
  return <>{formatTimer(duration)}</>; // HH:MM:SS format
};
```

## Flow Data Structure

Conversation flows retrieved from BuilderBot:

```typescript
interface Flow {
  id: string;
  name: string;
  label: string;
  keyword: string[];           // Trigger words
  answers: Answer[];           // Bot responses
  options: FlowOptions;        // Flow settings
  botId: string;
  // ... timestamps and metadata
}

interface Answer {
  message: string;             // Response text
  type: string;                // Message type
  plugins: Plugins;            // Integrations
  options: AnswerOptions;      // Response behavior
  // ... configuration
}
```

## Bot Operations

The Bots component handles these operations:

```typescript
// Fetch all bots
const data = await getBots();

// Create new bot
const newBot = await addBot(botData);

// Update bot configuration
const savedBot = await updateBot(botToUpdate);

// Delete bot
await deleteBot(botId);

// Connection management
await connectBot(botId);
await disconnectBot(botId);

// Retrieve bot data
const qrData = await getBotQrCode(id_builderBot);
const flows = await getBotFlows(id_builderBot);
```
